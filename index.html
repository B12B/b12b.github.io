Featured Project

Titanic++ — ML Survival Predictor

Capstone · Open Source

End-to-End Machine Learning System

A full production-grade ML system that turns raw Titanic passenger data into real-time survival predictions via a deployed Django web application — simulating the complete professional ML lifecycle from raw data to live product.

  • Engineered 4 novel features: Title extraction, FamilySize, IsAlone flag, and CabinDeck — each backed by EDA insights
